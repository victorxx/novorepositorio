🎮 História do Jogo Pong: O Início da Era dos Videogames

PRECISA DE UMA VPS PARA RODAR PROGRAMA OS SEUS RÔBOS NA NUVEM NOS TEMOS A SOLUÇÃO



QUERO COLOCAR PROPAGANDA AQUI

O Pong é considerado um dos primeiros jogos eletrônicos de sucesso comercial e um marco na história dos videogames. Embora seja um jogo simples — uma simulação de tênis de mesa com dois “paddles” e uma bola quadrada — ele ajudou a dar origem à indústria multibilionária dos games como conhecemos hoje.

QUER UMA PLATAFORMA SEGURA PARA FAZER SEUS ANÚNCIOS AQUI ESTÁ

🛠️ Como tudo começou

  • Ano: 1972
  • Criador: Allan Alcorn, engenheiro da empresa Atari
  • Fundador da Atari: Nolan Bushnell

Allan Alcorn foi contratado pela Atari, uma pequena startup de tecnologia fundada por Nolan Bushnell e Ted Dabney. Como um “teste” de suas habilidades, Bushnell pediu que Alcorn criasse um jogo simples baseado em tênis de mesa. O que era para ser um exercício interno, acabou se tornando um fenômeno!

🧪 Protótipo que deu certo

O primeiro protótipo do Pong foi instalado em um bar chamado Andy Capp’s Tavern, na Califórnia. Em poucos dias, a máquina travou — não por defeito, mas porque estava lotada de moedas. A popularidade foi imediata.

📈 O sucesso nas máquinas de arcade

Após o sucesso do protótipo, a Atari lançou o Pong comercialmente. Foi um enorme sucesso nos fliperamas (arcades). As máquinas estavam sempre ocupadas e os donos de bares e fliperamas faziam filas para comprar mais unidades.

⚖️ Polêmica com a Magnavox

A ideia de um jogo de tênis eletrônico já havia aparecido antes no Magnavox Odyssey, o primeiro console doméstico, lançado também em 1972. A empresa Magnavox processou a Atari por violação de patente. A disputa terminou com um acordo: a Atari pagou uma taxa e seguiu produzindo Pong.

🏠 Pong em casa

Em 1975, a Atari lançou uma versão doméstica do Pong, conectada diretamente à TV. Foi vendida pela loja Sears e virou febre no Natal daquele ano.
Esse foi o pontapé para a explosão dos videogames domésticos nos anos seguintes.

🕹️ Por que Pong é tão importante?

  • Foi o primeiro jogo a se tornar um sucesso comercial global
  • Popularizou os arcades
  • Provou que os videogames tinham potencial como indústria
  • Deu origem à Atari como gigante dos games

Se você está começando a programar em Python e quer criar seu primeiro jogo, o Pong é uma excelente escolha! 🕹️
É simples, divertido e ótimo para praticar conceitos como loops, colisões e controle de eventos com teclado.

Hoje, vamos te mostrar como criar o Pong clássico usando a biblioteca turtle do Python!

✅ Pré-requisitos

Você precisa apenas de:

  • Python instalado (versão 3.x)
  • Biblioteca padrão turtle (já vem com o Python)

🛠️ Passo a Passo: Código Completo

import turtle

# Janela do jogo
janela = turtle.Screen()
janela.title("Pong em Python")
janela.bgcolor("black")
janela.setup(width=800, height=600)
janela.tracer(0)

# Raquete esquerda
raquete_esquerda = turtle.Turtle()
raquete_esquerda.speed(0)
raquete_esquerda.shape("square")
raquete_esquerda.color("white")
raquete_esquerda.shapesize(stretch_wid=5, stretch_len=1)
raquete_esquerda.penup()
raquete_esquerda.goto(-350, 0)

# Raquete direita
raquete_direita = turtle.Turtle()
raquete_direita.speed(0)
raquete_direita.shape("square")
raquete_direita.color("white")
raquete_direita.shapesize(stretch_wid=5, stretch_len=1)
raquete_direita.penup()
raquete_direita.goto(350, 0)

# Bola
bola = turtle.Turtle()
bola.speed(0)
bola.shape("square")
bola.color("white")
bola.penup()
bola.goto(0, 0)
bola.dx = 0.2
bola.dy = 0.2

# Movimento das raquetes
def mover_cima_esquerda():
    y = raquete_esquerda.ycor()
    if y < 250:
        raquete_esquerda.sety(y + 20)

def mover_baixo_esquerda():
    y = raquete_esquerda.ycor()
    if y > -240:
        raquete_esquerda.sety(y - 20)

def mover_cima_direita():
    y = raquete_direita.ycor()
    if y < 250:
        raquete_direita.sety(y + 20)

def mover_baixo_direita():
    y = raquete_direita.ycor()
    if y > -240:
        raquete_direita.sety(y - 20)

# Teclado
janela.listen()
janela.onkeypress(mover_cima_esquerda, "w")
janela.onkeypress(mover_baixo_esquerda, "s")
janela.onkeypress(mover_cima_direita, "Up")
janela.onkeypress(mover_baixo_direita, "Down")

# Loop principal do jogo
while True:
    janela.update()

    # Mover a bola
    bola.setx(bola.xcor() + bola.dx)
    bola.sety(bola.ycor() + bola.dy)

    # Colisão com as bordas
    if bola.ycor() > 290:
        bola.sety(290)
        bola.dy *= -1

    if bola.ycor() < -290:
        bola.sety(-290)
        bola.dy *= -1

    # Fora do campo
    if bola.xcor() > 390:
        bola.goto(0, 0)
        bola.dx *= -1

    if bola.xcor() < -390:
        bola.goto(0, 0)
        bola.dx *= -1

    # Colisão com as raquetes
    if (340 < bola.xcor() < 350) and (raquete_direita.ycor() - 50 < bola.ycor() < raquete_direita.ycor() + 50):
        bola.setx(340)
        bola.dx *= -1

    if (-350 < bola.xcor() < -340) and (raquete_esquerda.ycor() - 50 < bola.ycor() < raquete_esquerda.ycor() + 50):
        bola.setx(-340)
        bola.dx *= -1
